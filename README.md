### About me

I'm a frontend / fullstack developer based in Germany, focused on building reliable, well-tested web applications and clean developer experience. Over the last several months I've been working closely with AI coding agents — primarily **Claude** and **Gemini** — integrating them into my day-to-day development workflow for architecture, refactoring and large-scale code reviews.

Outside of pure product work, I have a strong interest in **application security** and spend time researching static analysis, vulnerability detection and safe proof-of-concept generation for JavaScript / TypeScript codebases.
